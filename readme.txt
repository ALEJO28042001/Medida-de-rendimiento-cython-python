Para correr el programa se debe ejecutar en siguiente comando en una terminal que se encuntre en la dirección de los archivos
python3 setup.py build_ext --inplace
Ahora, se debe ingresar en el lanzador.pl y modificar la variable Path: Se introduce la dirección contenedora de los archivos

Luego se corre el lanzador: perl lanzador.pl

Se extrae y analiza la información
